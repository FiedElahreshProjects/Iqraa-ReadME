# Iqraa 📖✨  
*A Duolingo-style iOS app for Islamic learning*  

Iqraa is a mobile learning platform built in **Swift** with a **Supabase backend**, designed to make foundational Islamic knowledge accessible, engaging, and scalable. Inspired by Duolingo’s gamified approach, the app features **modular lessons, in-app subscriptions, and A/B testing** for continuous improvement.  

## 🚀 Features
- 📚 **Foundational Islamic Learning** – Bite-sized lessons delivered through a gamified experience.  
- 📱 **iOS App (Swift)** – Production-ready app architected for performance and scalability.  
- 🗄️ **Supabase Backend** – Real-time database and authentication to manage users and content.  
- 💳 **RevenueCat Integration** – Subscription support and payment infrastructure.  
- 🧪 **A/B Testing** – Optimized learning modules and onboarding funnels through experimentation.  
- 🌍 **Scalable Design** – Built with growth in mind, preparing for influencer-driven App Store launch.  

## 🛠️ Tech Stack
- **Frontend (iOS):** Swift, UIKit/SwiftUI  
- **Backend:** Supabase (Postgres, Auth, Functions)  
- **Payments & Subscriptions:** RevenueCat  
- **Experimentation:** A/B testing for onboarding and lesson design  
- **Other Tools:** GitHub, Xcode, TestFlight
